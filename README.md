## Data structure

- dat/      <-- all audio and transcript data
  - AIR/    <-- data from All India radio
    
- src/      <-- all Python scripts for running models
- res/      <-- project resources
  - codebook.txt (tab-separated file with phonemic transcription conventions between IPA and ASCII system)
